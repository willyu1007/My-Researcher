# Roadmap

## Objective
- Provide retrieval and trigger UX that can serve topic, paper-management, and writing workflows.

## Milestones
1. Retrieve contract.
2. Retrieve service.
3. Stale propagation.
4. Overview/detail UX.
5. Verification.

## Test And Acceptance Baseline
- Profile changes do not require reindexing.
- Stale state changes action availability but never enqueues automatically.
