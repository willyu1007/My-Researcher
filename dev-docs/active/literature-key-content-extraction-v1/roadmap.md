# Roadmap

## Objective
- Produce trustworthy, source-grounded semantic dossiers for downstream automation.

## Milestones
1. Schema and validator.
2. Extraction profiles and prompts.
3. Consolidation and source-ref validation.
4. Human override preservation.
5. Verification.

## Test And Acceptance Baseline
- READY/PARTIAL_READY/FAILED are deterministic from validator output.
- Invalid or ungrounded LLM output cannot mark the stage ready.
