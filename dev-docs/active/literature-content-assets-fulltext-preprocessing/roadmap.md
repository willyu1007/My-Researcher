# Roadmap

## Objective
- Provide durable fulltext and multimodal anchors for downstream semantic extraction and retrieval.

## Milestones
1. Inventory current artifact runtime.
2. Define artifact/asset contract.
3. Implement local asset-backed preprocessing.
4. Add fixture-driven verification.

## Test And Acceptance Baseline
- Every generated source-derived artifact has checksum and anchor metadata.
- Partial extraction is visible through diagnostics.
- Downstream source refs resolve.
